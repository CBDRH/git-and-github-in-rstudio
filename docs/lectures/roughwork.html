Create a new project in RStudio

  1. In the top right-hand corner of Rstudio (i) click the arrow and (ii) choose New Project
  2. Select New Directory then New Project
  3. Specify a directory name and subdirectory
  4. Tick Create a Git Repository

Mini-Exercise

3 minutes

Are you here yet?

  1. Enter here::here() at the console to see your current working directory. This should be where you saved your project!

  2. Enter here::here(".gitignore") at the console. What happens?

  3. Enter file.exists(here::here(".gitignore")) at the console. This should return TRUE. Note that you don’t need the full path to locate the file!

Note We will talk more about what the .gitignore file actually does later
